Technical Lead Engineer

Job Description:

Technical Lead Engineer to work on the full stack from transaction processing to building user interfaces.

Requirements:

  1. 3+ years experience in web application development and architecture to build a stable, high-performance platform, using Python and related technologies for new feature development.
  2. Experience with Flask or Django a must.
  3. Proficiency with MongoDB replication and sharding.
  4. Knows and can talk about web issues such as API design, security, scalability and data management.
  5. Strong problem-solving abilities and an attitude for learning new technologies.

A plus but not necessary:

  1. Sound knowledge of front-end development including CSS, Javascript and HTML5.
  2. Used Amazon EC2 to build a scalable product with Apache WSGI or similar application server.
